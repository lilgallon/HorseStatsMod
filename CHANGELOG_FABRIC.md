- Cloth config: `>=19.0.0`
- Fabric API: `*`

Suggestion:
- Modmenu: you will be able to edit the config with an interface

- [#30](https://github.com/lilgallon/HorseStatsMod/issues/30): adjusted min / max of every rideable entity
